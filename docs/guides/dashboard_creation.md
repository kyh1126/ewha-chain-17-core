# 대시보드 생성 가이드

이 가이드는 Looker Studio를 사용한 대시보드 생성 방법을 설명합니다.

## 빠른 시작

1. [Looker Studio 가이드](../../templates/dashboard/looker_studio_guide.md) 참고
2. BigQuery 데이터 소스 연결
3. 기본 차트 생성
4. 필터 및 스타일링

## 상세 가이드

자세한 내용은 `templates/dashboard/looker_studio_guide.md` 파일을 참고하세요.

## 대안: Next.js 대시보드

Looker Studio 대신 커스텀 웹 대시보드를 만들고 싶다면:

1. Next.js 프로젝트 초기화
2. BigQuery API 연동
3. 차트 라이브러리 사용 (Recharts, Chart.js 등)
4. Extension 트랙 A에서 상세 구현
