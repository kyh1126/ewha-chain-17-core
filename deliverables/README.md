# 최종 산출물

이 폴더는 W4 이후 최종 산출물을 제출하는 공간입니다.

## 제출 항목

### 필수 산출물

- [ ] **BigQuery SQL 쿼리 세트** (10~15개)
  - 위치: `members/[본인이름]/queries/`
  - 형식: `.sql` 파일

- [ ] **표준 대시보드**
  - Looker Studio 링크 또는 웹 URL
  - 위치: `members/[본인이름]/dashboard/`

- [ ] **Demo Video** (2~3분)
  - 내용: 문제 정의 → 대시보드 → Gemini 요약 → 결론
  - 위치: `deliverables/demo_videos/[본인이름]_demo.mp4`
  - 또는 YouTube 링크: `deliverables/demo_videos/[본인이름]_link.txt`

- [ ] **요약 리포트** (1~2p)
  - 위치: `deliverables/reports/[본인이름]_report.pdf`
  - 형식: PDF 또는 Markdown

- [ ] **실행 가이드**
  - 위치: `deliverables/guides/[본인이름]_guide.md`
  - 내용: 쿼리 실행 → 대시보드 확인 → 요약 생성

## 제출 방법

1. 각 산출물을 해당 폴더에 저장
2. `members/[본인이름]/README.md`에 링크 추가
3. PR 생성 시 "최종 산출물 제출" 태그 추가

## 평가 기준

- **완성도**: 필수 항목 모두 제출
- **품질**: 쿼리 정확성, 대시보드 가독성, 리포트 전문성
- **차별화**: 개별 지표 및 인사이트의 독창성
